Sticker United

Overview

Vision

We set out to build a back-end project that would be the skeleton for an e-commerce website.

Design

Design took a back seat in this project as the main objectives were to understand postgreSQL, tables, and bcrypt. The design we did decide on is reminiscent of the Supreme web store with minimal visual stimulus and wording on the landing page. A few notes about design:

  • We settled on a clear landing page with distinct pictures for each category
  • It was important to choose images that are easily recognizable so the audiences attention would be captured right away.
  • Lastly regarding design, having examples of the products on the landing page not only lets the user know what they can expect from each category, but also presents the most valuable imformation immediately.

One of our team members chose to manually create all the CSS from scratch instead of utilizing a framework. He wanted extra practice with design and this gave him that opportunity.

Results

Features

The team was able to create a solid beginning to an e-commerce site. Coincidentally, a member of our team actually sells custom stickers as a side hustle. Our hope is that he will be able to use this site in the future to break into the market. Here are some of the features within our code:

  • Login page with credentials protected by bcrypt
  • Ability to select any sticker as a favorite and add it to your profile
  • Ability to create an account
