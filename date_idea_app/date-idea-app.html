Date Idea App

Overview

Vision

My goal in this project was to create an app that could be used by people just like me. I have a hard time thinking of fun date ideas, and even when I do, often times I'm not sure where I could go to for the date idea I come up with. This app solves those problems.

Design

A clear and clean UI was important for me on this project. The emphasis is on the functionality. A few notes about design:

  • I wanted the UI to be unassuming (letting the user use it inconspicuously) if they needed to plan something as a surprise
  • There are three options to choose from, but only one per submit
  • After choosing one of the first search options, the other two submit buttons become disabled

A small amount of bootstrap was used to give the submit buttons a streamlined look.

Logistics

  • Development: June 16 - 25
  • My roles: Trivia page design, API implementation, project management

Results

Features

My team created a fun app that successfully utilized two different APIs. Here are some of the features within our code:

  • Added a 'Back' button to return to the landing page from the trivia page or the joke page
  • Wrote proper async/await functions to handle the APIs
  • We were able to randomly sort the answers on the page and let the user know when the correct answer was selected
