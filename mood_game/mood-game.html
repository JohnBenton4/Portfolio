Mood Game

Overview

Vision

We set out to build a front-end project that could be not only entertaining but also fun and informative.

Design

An important design factor was to keep the UI simple and give it a basic feel. The design emphasis is on the information on the page and has an 80's video game feel. A few notes about design:

  • Landing page needed to be simple and easy to understand
  • The use of theater faces, comedy and tragedy, is an homage to age old entertainment but in a twenty-first century web app

We used bootstrap due to its flexibility and ease of use.

Logistics

  • Development: June 16 - 25
  • My roles: Trivia page design, API implementation, project management

Results

Features

My team created a fun app that successfully utilized two different APIs. Here are some of the features within our code:

  • Added a 'Back' button to return to the landing page from the trivia page or the joke page
  • Wrote proper async/await functions to handle the APIs
  • We were able to randomly sort the answers on the page and let the user know when the correct answer was selected

Project Reflection

My group worked very well together. In only a few days, we were able to turn a rough idea into a fully functional game using API calls. This was my first project as a web developer. I learned so much not only from a strictly coding stand point, but also in terms of planning, communication, time management, and expectations.
